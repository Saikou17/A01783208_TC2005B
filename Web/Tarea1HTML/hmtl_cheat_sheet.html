Tags Principales:

a The Anchor Element

El Elemento HTML Anchor crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.

Ejemplo: Video de ejemplos del elemento a

p The Pharagraph Element

El elemento HTML p representa un párrafo. Los párrafos generalmente se representan en los medios visuales como bloques de texto separados de los bloques adyacentes por líneas en blanco y/o sangría en la primera línea, pero los párrafos HTML pueden ser cualquier agrupación estructural de contenido relacionado, como imágenes o campos de formulario.

Ejemplo: Video de ejemplo del elemento p

h1,h2...h6 The section Heading Elements

Los elementos HTML h1 a h6 representan seis niveles de encabezados de sección. h1 es el nivel de sección más alto y h6 es el más bajo.

Ejemplo: Video de ejemplo del elemento encabezado

div The Content Division Element

El elemento HTML div es el contenedor genérico para el contenido de flujo. No tiene ningún efecto sobre el contenido o el diseño hasta que se le aplica estilo de alguna manera usando CSS (por ejemplo, se le aplica estilo directamente o se aplica algún tipo de modelo de diseño como Flexbox a su elemento principal).

Ejemplo: Video de ejemplo del elemento div

span The Content Span Element

El elemento HTML span es un contenedor genérico en línea para redactar contenido, que no representa inherentemente nada. Se puede usar para agrupar elementos con fines de estilo (usando los atributos class o id), o porque comparten valores de atributo, como lang. Debe usarse solo cuando ningún otro elemento semántico sea apropiado. span es muy parecido a un elemento div, pero div es un elemento a nivel de bloque mientras que span es un elemento en línea.

Ejemplo: Video de ejemplo de elemento span

strong The Importance Element

El elemento HTML strong indica que su contenido tiene una gran importancia, seriedad o urgencia. Los navegadores normalmente muestran el contenido en negrita.

Ejemplo: Video de ejemplo de elemento strong

i The Idiomatic Text Element

El elemento HTML i representa un rango de texto que se separa del texto normal por alguna razón, como texto idiomático, términos técnicos, designaciones taxonómicas, entre otros. Históricamente, estos se han presentado en cursiva, que es la fuente original del nombre i de este elemento.

Ejemplo: Video de ejemplo de elemento i

u The Unarticulated Annotation Element

El elemento HTML u representa un tramo de texto en línea que debe representarse de manera que indique que tiene una anotación no textual. Esto se representa de forma predeterminada como un subrayado sólido simple, pero se puede modificar con CSS.

Ejemplo: Video de ejemplo del elemento u

b The Bring Attention To Element

El elemento HTML b se utiliza para llamar la atención del lector sobre el contenido del elemento, al que de otro modo no se le concede una importancia especial. Esto se conocía anteriormente como el elemento en negrita, y la mayoría de los navegadores aún dibujan el texto en negrita.

Ejemplo: Video de ejemplo del elemento b

strike The Line Element

El elemento HTML strike coloca un tachado (línea horizontal) sobre el texto.

Ejemplo: Video de ejemplo de elemento strike o linea

small The Side Comment Element

El elemento HTML small representa comentarios secundarios y letra pequeña, como derechos de autor y texto legal, independientemente de su presentación con estilo.

Ejemplo: Video de ejemplo de elemento small

em The Emphasis Element

El elemento HTML em marca el texto que tiene énfasis. El elemento em se puede anidar, y cada nivel de anidamiento indica un mayor grado de énfasis.

Ejemplo: Video de ejemplo de elementos em

header The Headers

El elemento HTML header representa contenido introductorio, normalmente un grupo de ayudas introductorias o de navegación. Puede contener algunos elementos de encabezado, pero también un logotipo, un formulario de búsqueda, el nombre del autor y otros elementos.

Ejemplo: Video de ejemplo de elemento header

section The Generic Section Element

El elemento HTML section representa una sección independiente genérica de un documento, que no tiene un elemento semántico más específico para representarla.

Ejemplo: Video de ejemplo de elemento section

article The Article Contents Element

El elemento HTML article representa una composición independiente en un documento, página, aplicación o sitio, que está destinado a ser distribuible o reutilizable de forma independiente (por ejemplo, en sindicación).

Ejemplo: Video de ejemplo de elemento article

aside The Aside Element

El elemento HTML aside representa una parte de un documento cuyo contenido solo está indirectamente relacionado con el contenido principal del documento.

Ejemplo: Video de ejemplo de elemento aside

footer Footer Element

Un footer normalmente contiene información sobre el autor de la sección, datos de derechos de autor o enlaces a documentos relacionados.

Ejemplo: Video de ejemplo de elemento footer

nav The Navigation Section Element

El elemento HTML nav representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o hacia otros documentos.

Ejemplo: Video de ejemplo de elemento nav

img The Image Embed Element

El elemento HTML img incrusta una imagen en el documento.

Ejemplo: Video de ejemplo de elemento img

br The Line Break Element

El elemento HTML br produce un salto de línea en el texto (retorno de carro). Es útil para escribir un poema o un discurso, donde la división de líneas es significativa.

Ejemplo: Video de ejemplo de elemento br

hr The Thematic Break Element

El elemento HTML hr representa una ruptura temática entre elementos a nivel de párrafo: por ejemplo, un cambio de escena en una historia o un cambio de tema dentro de una sección.

Ejemplo: Video de ejemplo de elemento hr

form The Form Element

El elemento HTML form representa una sección de documento que contiene controles interactivos para enviar información.

Ejemplo: Video de ejemplo de elemento form

table The Table Element

El elemento HTML form representa una sección de documento que contiene controles interactivos para enviar información.
