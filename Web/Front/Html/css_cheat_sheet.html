Border

La propiedad CSS abreviada de borde establece el borde de un elemento. Establece los valores de border-width, border-style y border-color.

Border-left, Border-right, Border-top, Border-bottom

Los border-left,right,top y bottom son utilizados para modificar los border izquierdo, derecho, arriba y abajo de manera independiente

Border Radius

La propiedad CSS border-radius redondea las esquinas del borde exterior de un elemento. Puede establecer un solo radio para hacer esquinas circulares o dos radios para hacer esquinas elípticas.

Background-image

Establece una imagen como fondo de pantalla

Color

La propiedad color CSS establece el valor de color de primer plano del texto y las decoraciones de texto de un elemento, y establece el valor de color actual.

Font

La propiedad abreviada CSS de fuente establece todas las diferentes propiedades de la fuente de un elemento. Alternativamente, establece la fuente de un elemento en una fuente del sistema.

Font-Family

La propiedad CSS font-family especifica una lista priorizada de uno o más nombres de familia de fuentes y/o nombres de familia genéricos para el elemento seleccionado.

Font-Style,Font-Weight

El estilo y peso de la fuente son modificados por estos selectores.La propiedad CSS de estilo de fuente establece si una fuente debe tener un estilo normal, cursiva u oblicua de su familia de fuentes.La propiedad CSS font-weight establece el peso (o negrita) de la fuente.

Width,Height

La propiedad CSS ancho establece el ancho de un elemento. De forma predeterminada, establece el ancho del área de contenido, pero si el tamaño del cuadro se establece en border-box, establece el ancho del área del borde.La propiedad CSS de altura especifica la altura de un elemento. De forma predeterminada, la propiedad define la altura del área de contenido.

List-Style

La propiedad abreviada CSS de estilo de lista le permite establecer todas las propiedades de estilo de lista a la vez.

Margin

La propiedad abreviada margin CSS establece el área de margen en los cuatro lados de un elemento.

Padding

La propiedad abreviada padding CSS establece el área de relleno en los cuatro lados de un elemento a la vez.

Overflow

La propiedad abreviada CSS de desbordamiento establece el comportamiento deseado para el desbordamiento de un elemento, es decir, cuando el contenido de un elemento es demasiado grande para caber en su contexto de formato de bloque, en ambas direcciones.

Display

La propiedad CSS de visualización establece si un elemento se trata como un bloque o un elemento en línea y el diseño utilizado para sus elementos secundarios, como el diseño de flujo, la cuadrícula o la flexión.

Position

La propiedad position CSS establece cómo se coloca un elemento en un documento. Las propiedades superior, derecha, inferior e izquierda determinan la ubicación final de los elementos posicionados.

Filter

La propiedad CSS del filtro aplica efectos gráficos como desenfoque o cambio de color a un elemento. Los filtros se usan comúnmente para ajustar la representación de imágenes, fondos y bordes.
